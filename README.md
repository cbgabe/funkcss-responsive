## Responsive
Funkcss extends [funkcss](https://github.com/filipelinhares/funkcss) to be responsive.

## Development
```
npm install
gulp watch
```
**Other gulp comands**

- `gulp postcss` - compile lib to dist
- `gulp min` - minify dist
- `gulp` - both together
